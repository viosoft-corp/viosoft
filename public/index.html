  • Driving digital transformation
    From edge devices to core network
    and between

We deliver products and services to help customers build and operate unified, end-to-end digital infrastructure to collect, analyze, and deliver insight from real-time data in order to increase efficiency,  streamline automation, and accelerate business innovation.

IoT & Embedded

Development and debug tools for embedded micro-controllers, embedded Linux, fixed & mobile IoT, and telco-cloud.  Reference designs for consumer multimedia, field data acquisition and analysis, and real-time device monitoring and reporting.

Mobile Edge

Hyper-connected, low-latency eNB with high UE density.  Web-scale FlexRAN deployment with virtualized LTE network utilizing industry standard orchestration and automation framework.  Micro network application services for different wireless backend.

Access Network

Combine the pervasiveness of public network with the flexibility of software-defined networking to build private virtual mesh access network that meets or exceeds the performance and reliability standard set by traditional WAN deployment.

Core Network

Chaining of network applications to deliver new and innovative services, while aggregating compute-intensive use cases in the core network to optimize OpEx and CapEx. Production ready reference core network services such as router, firewall, DPI, and IPS.

Public & Private Cloud

Build, manage and operate private cloud with off-the-shelf unified end-to-end digital infrastructure (Kubernetes, OpenStack). Network slicing PoCs fast-tract deployment by reducing latency for critical applications while improving utilization of shared resources.

CI & CD / DevOps

Create and deploy agile network services on multi, hybrid clouds.  Integration platform to develop, test, debug, and deploy virtualized applications and micro services in continuous cycle with cutting edge DevOps and network service modeling tools.

Unified native services, with open infrastructure, on many clouds
